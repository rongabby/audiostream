Music Player - Iframe Integration Demo

This demonstrates how to embed the React Native web app in an iframe

Integration Instructions

1. Build the app for web:

npx expo export --platform web

2. Host the built files:

Upload the contents of the dist folder to your web server

3. Embed using iframe:

<iframe src="https://your-domain.com/path-to-app" width="100%" height="800px" frameborder="0"></iframe>

4. Customize dimensions:

Adjust width and height to fit your existing page layout

Configuration Notes

  • The app requires Supabase configuration for full functionality
  • Admin mode allows file uploads and playlist management
  • Visitor mode shows public playlists
  • The iframe includes necessary permissions for audio playback
  • Responsive design adapts to different screen sizes
